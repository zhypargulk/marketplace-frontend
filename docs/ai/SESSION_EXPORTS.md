# Экспорты AI-сессий (spec §8.1)

Сюда помещать экспорты AI-сессий (markdown, jsonl или текст) — столько, сколько уместно для понимания процесса.

**Примеры файлов в этом репо:**
- `session-export-sample.md` — детали продукта (seller, сортировка) и Admin UI (CRUD продукта/офферов, загрузка изображения).
- `session-export-fsd-auth.md` — FSD-слои и useAuth для синхронизации состояния логина.

**Имена файлов:** `session-YYYY-MM-DD.md`, `session-*.jsonl`, или осмысленные имена.
